# overview
Medical Encounter Management System (MEMS) is a FastAPI-based backend for managing patients, doctors, and medical appointments.\
It enforces business rules, timezone-aware datetime handling, and follows production-grade practices with CI, security checks, and testing.

# how to install dependencies
poetry install dependencies \
poetry install -dev dependencies

# how to run
unvicorn src.main:app --reload
